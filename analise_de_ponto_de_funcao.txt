ALI - Função de dados (interno): Grupos de dados logicamento correlatos, identificável pelo usuário,
mantido dentro da fronteira da aplicação.
	- Apresentar uma unidade lógica reconhecível peolo usuário
	- Ser mantido através de um processo elementar da aplicação

AIE - Função de dados (externo): Grupo de dados logicamente correlatos, identificável peolo, usuário
mantido por outra aplicação
	- Identificavél pelo usuário
	- Referenciado, mas mantido na fronteira de outra aplicaçãp
	- Armazenar dados
	- Consultados em um ou mais processos elementares da aplicação contada

Funções transacionais: Transações do sistema (leiturua e escrita)
	Entra Externa (EE): Um processo elementar, no qual dados cruzam a fronteira da aplicação
de fora para dentro com o objetivo de alterar o comportamento da aplicação ou dados
	
	Saída Externa (SE): Processo elementar de dentro para fora, através de uma lógica de proce
ssamento.

	Consulta Externa (CE):  Processo elementar, os dados vão de dentro para fora, sem envolver
cálculos ou alterações de dados
